
# Rainfall Weather Analysis Project

This project demonstrates a complete rainfall analysis and prediction pipeline using Python and MATLAB.

## Features
- Rainfall data preprocessing
- Exploratory data analysis
- Visualization
- Machine learning rainfall prediction
- MATLAB statistical rainfall modeling
- Modular project structure

## Folder Structure

data/ – rainfall datasets  
src/ – Python source code  
src/models/ – ML rainfall prediction models  
src/visualization/ – plotting and charts  
matlab/ – MATLAB rainfall analysis scripts  
config/ – configuration files  
docs/ – documentation  

## Running the Python Pipeline

1. Install requirements:
```
pip install pandas numpy matplotlib scikit-learn
```

2. Run analysis:

```
python src/main.py
```

## MATLAB

Open MATLAB and run:

```
run_rainfall_analysis.m
```

## Output

The program produces rainfall trend statistics and prediction results.
